# VRChat-players-blacklist

**A custom blacklist of VRChat players.**

</div>

---

### Introduction

This is a repository with a custom blacklist of VRChat players and a script to add them.

---

### About creating a blacklist

There are bad players in games (like VRChat), so I created this repository.

For ease of use in programs, I created a blacklist in the **json** format.

---

### Script usage

1. Run the script.
2. Follow the script instructions.
3. Save the blacklist data by pressing Ctrl+C.

> [!WARNING]
> If you do not enter complete information about the last player, he will not be recorded.

> [!CAUTION]
> **By the way, banning players from any lists through the use of certain programs is prohibited by the VRChat ToS.**

---

### About filing an appeal

You need to submit an appeal in **Issues** of this repository.

> [!CAUTION]
> **You can only appeal one player once!**
>
> **Do not create an incomplete appeal, otherwise you will receive a warning; if you create such an appeal again, you will be blocked.**
>
> **Multiple appeals about one player or offtopic will lead to them being ignored and the user who submitted them being blocked.**

---

### Rules for filing an appeal:

1. Create an issue in this repository.
2. Write the title "Appeal to remove a player from the blacklist".
3. Select the "**appeal**" label.
4. First, **write the player ID and the reason for being blacklisted**.
5. Write, why this player did what he was for blacklisted.
6. **Check the correct spelling of issue** and submit it.
